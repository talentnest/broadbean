# Broadbean
Communicate to Broadbean's AdCourier service by sending commands and parsing responses.

## Installation

Add this line to your application's Gemfile:

    gem 'broadbean'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install broadbean
